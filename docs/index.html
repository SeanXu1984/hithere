Hi There - 近くの人との匿名グループチャット

周りの人と完全に匿名でプライベートにチャットできます。インターネット不要、データ収集なし、サーバー不要です。

主な機能

登録不要

すぐにチャットを開始できます。メール、電話番号、アカウント作成は不要です。アプリを開いて始めるだけです。

完全匿名

接続を選択するまで身元を明かさずに近くのユーザーとチャットできます。匿名性は常に保護されます。

匿名グループチャット

周りの人とのグループ会話に参加しながら、完全に匿名のままです。

データ収集ゼロ

プライバシーを最優先にしています。Hi Thereは個人情報を収集、保存、送信しません。すべてがデバイスに保存されます。

オフラインで動作

インターネット接続不要！Hi Thereはローカル無線技術を使用して、すぐ近くの人とつながります。

リアルタイムメッセージング

近くのユーザーとの即座のコミュニケーション。メッセージはリアルタイムで表示され、ユーザーが範囲外に出ると消えます。

日常の場所に最適

オフィス 職場のゴシップを共有したり、仕事の話題を話し合ったり、休憩中に同僚とつながったりできます。
公共交通機関 通勤がもっと楽しくなります！地下鉄、バス、電車、路面電車で他の乗客とチャットできます。
大学キャンパス 教室、カフェテリア、図書館、キャンパスのどこでも学生とつながれます。
企業のカフェテリア ランチ休憩中に同僚と交流を始めましょう。

プライバシー第一

Hi Thereは、プライバシーを核心に構築されています。会話、プロフィール情報、すべてのデータはデバイスに残ります。追跡しません、データを保存しません、情報を誰とも共有しません。シンプルです。
