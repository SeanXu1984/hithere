# Hi There - GitHub Pages Website

This repository contains the GitHub Pages website for the Hi There app.

## Structure

All website files are located in the `docs/` folder. This allows the repository to serve as both a documentation source and a GitHub Pages site.

## GitHub Pages Setup

1. Go to repository Settings â†’ Pages
2. Under Source, select:
   - Branch: `main` (or `master`)
   - Folder: `/docs`
3. Save

The website will be available at: `https://seanxu1984.github.io/hithere/`

## Files

- `docs/index.html` - Home page
- `docs/privacy.html` - Privacy policy page
- `docs/support.html` - Support page
- `docs/styles.css` - Main stylesheet
- `docs/i18n.js` - Internationalization system
- `docs/script.js` - Main JavaScript
- `docs/translations/` - Translation files for multiple languages

## Features

- Multi-language support (Japanese, English, Traditional Chinese, German, Korean)
- Responsive design
- Privacy Policy page
- Support page with contact information
